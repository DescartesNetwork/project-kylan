initializeCert

  • initializeCert(printerAddress: string, secureTokenAddress: string, taxmanAuthorityAddress: string, rate: BN, fee?: BN): Promise<{ certAddress: string; txId: string }>
    • Defined in lib/index.ts:327

    Create a certificate that allows the printer could print/burn a new secure token. The ratio between the secure and stable token is defined by rate. 1 stable_token (in decimals) = 1 secure_token (in decimals) * rate / 10^6

    rate = price_of_secure_token * 10^(12 - secure_token_decimals) (This formula is hard set to stable with decimals 6)

    Parameters

    • printerAddress: string

      Printer address.

    • secureTokenAddress: string

      Secure token address.

    • taxmanAuthorityAddress: string

      Taxman authority (owner) address.

    • rate: BN

      Secure token rate with decimals 6.

    • fee: BN = ...

      Burning Fee. Default is 5000 with decimals 6 (0.5%).

    Returns Promise<{ certAddress: string; txId: string }>

initializePrinter

  • initializePrinter(printer?: Keypair, decimals?: number, stableToken?: Keypair): Promise<{ printerAddress: string; stableTokenAddress: string; txId: string }>
    • Defined in lib/index.ts:284

    Initialize a printer. This printer will generate & manage a stable token. After the init, the authority can add certificate to accept the pair of secure token & stable token. An number of secure tokens, which is verified by a certificate, is locked in the printer, a corresponding number of stable tokens would be printed and sent back.

    Parameters

    • printer: Keypair = ...

      (Optional) The printer keypair. If it's not provided, a new one will be auto generated.

    • decimals: number = 6

      (Optional) Decimals for the stable token. Default is 6.

    • stableToken: Keypair = ...

      (Optional) The stable token keypair. If it's not provided, a new one will be auto generated.

    Returns Promise<{ printerAddress: string; stableTokenAddress: string; txId: string }>
